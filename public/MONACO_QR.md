## How to Use

### Basic Usage
1. **Select problem** from dropdown
2. **Code loads** into editor
3. **Edit with features**
4. **Ctrl+Enter** to run

### Advanced Usage
- **Ctrl+Space** - Get code suggestions
- **Ctrl+F** - Find in code
- **Ctrl+H** - Replace text
- **Ctrl+G** - Go to line
- **Tab** - Indent (4 spaces)
